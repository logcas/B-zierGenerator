# BézierGenerator
贝塞尔曲线生成器

## cssBezier
通过可视化摆动贝塞尔曲线获取CSS的`transition`效果。

在线演示：http://139.199.14.123/cssbezier/

## VisualBezier
展示你喜欢的贝塞尔曲线生成过程。

#### Log
* 2018/12/27 - 用 TypeScript 重写并进行了封装。TypeScript版使用gulp构建。
* 2019/03/02 - TypeScript版使用叠层Canvas将曲线和直线分开渲染，解决卡顿和曲线不直问题。（在线演示没有更新）
